序言
*******
Go_ 是一帮 Google_ 天才员工以开源形式在许多人贡献帮助下开发的一种编程语言。
他的代码是开源的，你可以自由的预读和修改。

这本书尝试指导你如何有趣的去使用 Go 写代码。
为了让学习更加简单易懂，本书包含了许多的说明性的例子和图表。

为什么使用 Go?
=======
现在已经有一大堆（实际上并没那么多）的编程语言存在。
因此也许有人会有疑问，为什么还要创造另一种编程语言？
因为实际上，Go 是非常容易学习、非常有趣、效率非常高并且设计得非常好。

Go 是一种编译语言。
Go 是一种能直接以机器代码的形式直接运行的语言，
就是说他不同于 Python、Ruby、Perl 那样需要外部编译器解析才能运行起来。

实际上，如果我们需要拿 Go 和其他对比的话，那么他可能就是现代的 C 语言。
是的，主要依据就是因为他的强类型。
作为一个 C 语言程序员，C 语言在我心中有很重要的位置，这个位置重要到无法与其他东西共存。
但 Go 赢得了我的尊重，这本书重点会演示 Go 的各种简单、高效的特性以及各种先进的概念。

Go 带来了一些新的语法和用语，但这并不会像其他语言那样多得把学习者吓怕。不过也足够让语言有所进步。
这种平衡-我不得不说- *最简艺术*使得 Go 学习如此有趣。

Go 是非常高效率的静态语言，但它也提供了一些简单的语法和概念，这使得它如动态语言一样酷炫。

Go 程序编译和运行都非常快。同时它原生就支持协程和消息通信。

这本书的来由?
==============
介绍 Go 语言是份很舒适的工作。
这就是说对所有人来说都是很容易理解的，并且学习起来非常有趣和简单。
是的，这非常有挑战性，但是... *aude a liquid signum* [#f1] .

我个人不会接受 *"努力学习"* 的方式。学习不应该是 *困难的*！
只要记得我们幼小无知的时候是如何去学习说话的。
而 Go 语言以及其他语言，比人类语言简单多了； 这里更少的单词，更少的语法、规则、习语...

因此通过这本书学习的规则及唯一条件就是：轻轻松松地把他当成一个游戏。

我们会和数据及代码一起游戏，每次学点东西。
这里会从简单开始慢慢开放式丰富，甚至从一些愚蠢的东西开始到严格的、明智的概念。

本书提供的几乎所有的代码都可以通过简单的复制粘贴到 `Go Playground` 就可以进行在线测试。
